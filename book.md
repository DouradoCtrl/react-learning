# Componente

> Regra jsx: Não posso retornar mais de um elemento no return

**O que é state ?**
State: Variável que armazena um valor e pode ser alterada

**O que é hook ?**
Uma função especial que permite utilizar algumas funcionalidade especiais do react como o estado

```jsx
const [message, setMessage] = useState("Olá, mundo"); // Hook
// message é o valor incial (valor padrão)
// setMessage valor após a mundaça de estado

//componente renderizado novamente
```
